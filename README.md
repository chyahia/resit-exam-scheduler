# 📅 نظام مُجدوِل الامتحانات الاستدراكية (Resit Exam Scheduler)

تطبيق ويب تفاعلي متقدم مبني بلغة **Python** وإطار عمل **Streamlit**، يهدف إلى أتمتة وبناء جداول الامتحانات الاستدراكية (التعويضية) للجامعات والمؤسسات التعليمية بذكاء. يراعي النظام تعقيدات الجدولة الزمنية والمكانية، ويقدم حلولاً رياضية للمفاضلة بين راحة الأساتذة ومصلحة الطلبة.

---

## ✨ المميزات الرئيسية (Key Features)

* **🧠 خوارزميات جدولة ذكية:**
  * يدعم خوارزمية المسار الواحد (Greedy Algorithm) للتوزيع السريع.
  * يدمج محرك بحث عميق (Large Neighborhood Search - LNS) للبحث عن الحلول المثلى وتفادي الفجوات الزمنية.
* **⚖️ المفاضلة متعددة الأهداف (Multi-Objective Optimization):**
  * **استراتيجية الطالب:** إعطاء الأولوية لتتابع امتحانات المستوى الواحد لتفادي تشتت الطلبة.
  * **استراتيجية الأستاذ:** القتال خوارزمياً لتجميع مواد الأستاذ في نفس الحصة، أو نفس اليوم، أو أيام متتالية.
* **🛡️ نظام قيود صارم ومرن:**
  * يمنع تعارض المستويات (الامتحانات المتزامنة لنفس الدفعة).
  * يدير قيود الأساتذة (تفضيل التجميع، تجنب الحصة الأولى، ثنائيات المرافقة أو الانفصال).
  * توحيد القاعات تلقائياً للأستاذ الذي يشرف على أكثر من مادة في نفس الفترة الزمنية.
* **💾 إدارة متكاملة للبيانات:**
  * نظام قواعد بيانات محلي خفيف وآمن باستخدام `SQLite`.
  * إمكانية إضافة، تعديل، وحذف (الأساتذة، القاعات، المستويات، المواد) مع ميزة "التنظيف العميق" (Cascade Delete) لمنع الأخطاء الشبحية.
* **📄 تصدير احترافي:**
  * توليد الجداول النهائية بصيغة **Word (`.docx`)** مفصّلة (جداول خاصة بالمستويات وأخرى بالأساتذة) وجاهزة للطباعة مباشرة.
* **🎨 واجهة مستخدم عربية:**
  * تصميم جذاب ومتجاوب بالكامل (RTL) مع خط "تجوّل" (Tajawal) المدمج.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **لغة البرمجة:** Python 3.x
* **واجهة المستخدم:** Streamlit
* **قاعدة البيانات:** SQLite3
* **مكتبات إضافية:**
  * `python-docx` (لتوليد ملفات الوورد).
  * `streamlit-keyup` (للبحث الحي والتفاعلي داخل الواجهة).

## 🚀 طريقة التثبيت والتشغيل (Installation & Usage)

1. **استنساخ المشروع (Clone the repository):**
   ```bash
   git clone [https://github.com/chyahia/resit-exam-scheduler.git](https://github.com/chyahia/resit-exam-scheduler.git)
   cd resit-exam-scheduler

2.تثبيت الحزم والمكتبات المطلوبة:
python -m pip install streamlit python-docx streamlit-keyup

3.تشغيل التطبيق:
python -m streamlit run app.py
سيتم فتح التطبيق تلقائياً في متصفحك الافتراضي على الرابط: http://localhost:8501

📋 نظرة على مراحل العمل في النظام
1.إدخال البيانات: تسجيل الأساتذة، القاعات، والمستويات.

2.الحذف والتعديل: إدارة قاعدة البيانات بمرونة.

3.إسناد المواد: ربط المواد بالأساتذة بواجهة تفاعلية سريعة.

4.تخصيص القاعات: تحديد القاعات المتاحة لكل مستوى.

5.الأيام والأوقات: بناء الهيكل الزمني لفترة الامتحانات.

6.القيود: إعداد شروط الخوارزمية (تعارضات، تفضيلات، مرافقات).

7.التوزيع والتصدير: اختيار الاستراتيجية، تشغيل الخوارزمية، وتحميل ملفات الوورد النهائية.

تم تطوير هذا المشروع لتسهيل الإدارة الأكاديمية وحل معضلة الجداول المعقدة بأناقة وفعالية.
